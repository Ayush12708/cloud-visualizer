# ☁ Cloud Visualizer Dashboard

A dynamic cloud monitoring dashboard built using Django to visualize and manage cloud resources.

---

## 🚀 Features

- 📊 Interactive Dashboard UI  
- 📈 CPU Usage Graph (Chart.js)  
- 💰 Cost Analysis Graph  
- 🖥 Resource Table (VM status)  
- 🔄 Dynamic Data Rendering (Backend → Frontend)  
- 🎛 Start/Stop VM simulation  

---

## 🛠 Tech Stack

- **Backend:** Django (Python)  
- **Frontend:** HTML, CSS, JavaScript  
- **Charts:** Chart.js  
- **Version Control:** Git & GitHub
